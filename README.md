# Projeto Prático: Dominando a Linguagem de Programação Swift

## Sobre o Projeto

Este projeto faz parte da formação iOS Developer da DIO. O objetivo é dominar os conceitos de Programação Orientada a Objetos em Swift, aplicando conhecimento em estruturas de dados, controle de fluxo, funções avançadas e más práticas de programação em Swift.

## Estrutura do Projeto

### Projeto 1: Fundamentos Swift
- Conceitos básicos da linguagem
- Variáveis e constantes
- Tipos de dados
- Controle de fluxos
- Funções
- Coleções (Arrays, Dictionaries, Sets)

### Projeto 2: Programação Orientada a Objetos
- Classes e Objetos
- Propriedades e Métodos
- Herança
- Polimorfismo
- Encapsulamento
- Closures
- Protocolos

## Tecnologias Utilizadas

- **Linguagem**: Swift 5.0+
- **IDE**: Xcode 12.0+
- **Plataforma**: iOS

## Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/tilucasoliveira/Projeto-Swift-DIO-tilucasoliveira.git
cd Projeto-Swift-DIO-tilucasoliveira
```

2. Abra no Xcode:
```bash
open Fundamentos.playground
open POO.playground
```

3. Execute o código no Playground
